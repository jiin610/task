# 프로젝트 제안서
### 1. 프로젝트 이름
나만의 빈출 단어장(가제) 
### 2. 한 문장 설명
사용자가 추가한 텍스트 파일을 바탕으로, 빈출 단어를 구해 사용자 전용의 단어장을 생성한다.
### 3. 해결하고자 하는 문제 또는 개발 목적
외국어 공부를 할 때 어떤 단어부터 외워야 할지 막막한 경우가 많다.
가사나 책 같은 실제 텍스트에서 자주 나오는 단어를 먼저 익히면
훨씬 효율적으로 공부할 수 있다.
하지만 이를 수작업으로 하려면 시간이 너무 많이 걸린다.
텍스트 파일을 자동으로 분석해 고빈도 단어만 추려 외국어를 더 효율적으로 공부하고자 이 프로그램을 고안했다.
### 4. 핵심기능
1. **텍스트 전처리** — 언어 자동 감지 후 특수기호·공백을 제거하고 분석 가능한 형태로 정제
2. **빈도 통계 분석** — 단어별 등장 횟수와 상대 빈도(%) 산출 후 순위 정렬
3. **단어장 자동 출력기** — 상위 N개 고빈도 단어를 CSV 파일로 저장
### 5. 예상 사용자
영어 혹은 중국어를 공부하고 있는 사람
### 6. 결과물 형태
CLI(예상)
### 7. 사용할 파이썬 개념 
변수, 딕셔너리, 리스트, 함수, 반복문, 조건문, 문자열 처리
